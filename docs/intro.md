# Selection Manager (v1.3)
**Selection Manager** is a runtime object-selection toolkit for Unreal Engine — everything you need to let players select gameplay objects with the mouse, the way they do in RTS and strategy games. Click a unit, drag a selection box over a crowd, add to or subtract from the current selection, and group your units. <br>

It's **Blueprint-first** (no C++ required) and works with any `Actor`, `Pawn`, `Character`, or scene component — units, buildings, props, mixed together. <br>

Plugin on Fab: **[https://www.fab.com/listings/60701332-bc86-48ea-88ac-c13a145bf747](https://www.fab.com/listings/60701332-bc86-48ea-88ac-c13a145bf747)** <br>
