# Changelog

### Update v1.1 (only for version 5.4+)
- Fixed the ability to select unregistered units that have an interface using the cursor. <br>
- Added a setting to disable interaction of units with the cursor (selection and highlighting). <br>
 

### Update v1.2 (only for version 5.6+)
- Improved stability <br>


### Update v1.3 (only for version 5.7+)
- Plugin integration has been simplified <br>
- Cached selection state to reduce repeated Blueprint interface calls <br>
- Added selection query functions: Get Selected Actors Of Class, Get Selected Actors, etc. <br>
- Added functions for forcing object selection state <br>
- Optional marquee highlight throttling for optimization <br>
- Experimental world-space Spatial Grid optimization only for simple marquee collision checks <br>
- Added Simple Collision Offset per registered object <br>
- Added an option to disable highlight blinking when clicking on a selected object <br>
- Selection now works more efficiently with large numbers of marquee simple-collision units <br>
- Formation selection <br>
- Improved stability <br>
- Debugger fixes <br>
- Updated internal logic to avoid blocking multiplayer integration. Multiplayer support has not been fully tested yet, but integration should now be possible without major issues. <br>
